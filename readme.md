# 🎮 Pokedex

**Ein interaktiver Pokédex, der Daten von der [PokéAPI](https://pokeapi.co/) abruft und Pokémon in einer responsiven Webanwendung anzeigt.**
*Erstellt als Abgabeprojekt für die Weiterbildung zum Frontend-Web-Developer bei der Developer Akademie.*

---

## 📌 **Funktionen**
✅ **Anzeige von Pokémon-Karten** (20–40 pro Seite)
✅ **Lazy-Loading** für effizientes Laden von Daten (z. B. Evolution-Chains erst beim Klick)
✅ **Dynamische Suchfunktion** (ab 3 Buchstaben, mit Fehlerfeedback)
✅ **Responsive Design** (optimiert für 320px bis 1920px)
✅ **Detailansicht (Overlay)** mit:
   - Name, Typ(en), Bild
   - Stats (HP, Attack, Defense etc.)
   - Navigation zwischen Pokémon (Pfeile)
✅ **Loading-Screen** und deaktivierter "Load More"-Button während des Ladens
✅ **Caching** von API-Daten zur Vermeidung redundanter Anfragen
✅ **Farbige Hintergrundkarten** basierend auf Pokémon-Typen

---

## 🛠 **Verwendete Technologien**
- **HTML5** (semantische Struktur)
- **CSS3** (Flexbox, Grid, Responsive Design)
- **JavaScript (Vanilla)** (Fetch API, DOM-Manipulation, Event-Listener)
- **PokéAPI** ([pokeapi.co](https://pokeapi.co/)) für Pokémon-Daten
